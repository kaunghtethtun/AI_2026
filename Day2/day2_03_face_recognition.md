#### RAW TO TRAINING DATA
- face recognition မပြောခင် math လေးနဲနဲပြောမယ် symmetric matrix ကို သိလား
- transport လုပ်ရင် ဒါပဲပြန်ရတဲ့ကောင် , ကြိုက်ရာ matrix တခုကိုယူပြီး သူ့ရဲ့ transpose နဲ့ မြှောက်ရင် symmetric matrix ရတယ်။
- eigen ပါလာမယ် ရုတ်တရက် နဲနဲ မြင့်သွားမယ်။ ရပါတယ် တွက်နည်းကို အာရုံမစိုက်နဲ့ နားလည်အောင် ပဲလေ့လာပါ။
- နောက် vector တစ်ခုရှိတယ်ဆိုပါစို့၊ 2 row, 1 column ထားပါတော့။ [ 3 4] 
- သူ့ကို 2x2 matrix နဲ့မြှောက်မယ်ဆိုရင် [0 1, 1 0] , ဒါက လူတိုင်းသိပေမဲ့ အခြား အမြင်တွေရှိဖို့လိုအပ်သေးတယ်။
- မြင်ဖို့က vector တစ်ခုကို matrix တစ်ခုနဲ့ apply လုပ်တဲ့အခါ [ 4 3] ဖြစ်တယ်။ input , output ပေါ့။
- နောက်တချက်မြင်ရမှာက အဲ့ဒီ matrix က ဖန်ရှင်လို့မြင်သင့်တယ်။ ဘာလို့ဆို output ထွက်လာလို့
- နောက် domain change သွားတယ်လို့လဲ ပြောလို့ရတယ်။ 
- transform ဖြစ်သွားတယ်။ dimension တူ၊ တိုး၊ လျော့ သွားတာလည်းဖြင်နိုင်တယ်.
- ဒါကြောင့် matrix တစ်ခုနဲ့ မြှောက်တာကို ဒါမျိုး အစုံ မြင်တဲ့ အမြင်ရှိဖို့လိုတယ်။ ဒါတွေက သင်ချာသင်တဲ့ အခါ ပြပါမယ်။
