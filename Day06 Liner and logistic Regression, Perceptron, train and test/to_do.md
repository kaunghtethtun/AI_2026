### သင်တန်းသားများလေ့ကျင့်ရန်

#### Linear Regression
- [[ House Prices - Advanced Regression Techniques Dataset ]](https://www.kaggle.com/datasets/vikrishnan/boston-house-prices): Linear Regression အတွက် နာမည်ကြီးဆုံး Dataset ပါ။ အိမ်ရဲ့ အကျယ်အဝန်း၊ တည်နေရာပေါ်မူတည်ပြီး ဈေးနှုန်းကို ခန့်မှန်းရမှာပါ။

- [[ Medical Cost Personal Dataset ]](https://www.kaggle.com/datasets/mirichoi0218/insurance): အသက်အရွယ်၊ BMI နဲ့ ဆေးလိပ်သောက်ခြင်းတို့ပေါ် မူတည်ပြီး ကျန်းမာရေး အာမခံစရိတ် ဘယ်လောက်ကျသင့်မလဲဆိုတာကို တွက်ချက်ရမှာပါ။

- [[ Advertising Dataset ]](https://www.kaggle.com/datasets/ashydv/advertising-dataset): TV, Radio နဲ့ Newspaper တွေမှာ ကြော်ငြာခသုံးစွဲမှုအပေါ် မူတည်ပြီး Sales (အရောင်းသွက်မှု) ကို ခန့်မှန်းတဲ့ နေရာမှာ သုံးပါတယ်။

- [[ Auto MPG Dataset ]](https://www.kaggle.com/datasets/uciml/autompg-dataset): ကားရဲ့ အင်ဂျင်ပါဝါ၊ အလေးချိန် စတာတွေအပေါ် မူတည်ပြီး လောင်စာဆီစားနှုန်း (Miles per gallon) ကို ခန့်မှန်းဖို့ပါ။

- [[ Salary Prediction Dataset ]](https://www.kaggle.com/datasets/rkiattisak/salaly-prediction-for-beginer): အလုပ်အတွေ့အကြုံ (Years of Experience) အပေါ် မူတည်ပြီး လစာကို ခန့်မှန်းတဲ့ ရိုးရှင်းတဲ့ Dataset ဖြစ်ပါတယ်。

- [[ Real Estate Price Prediction Dataset ]](https://www.kaggle.com/datasets/quantbruce/real-estate-price-prediction): တိုင်ပေမြို့က အိမ်ခြံမြေဈေးနှုန်းတွေကို တည်နေရာနဲ့ အနီးနားက အဆောက်အဦးတွေအပေါ် မူတည်ပြီး ခန့်မှန်းရမှာပါ။

#### နမူနာ code
- [[ Fish Market Dataset ]](https://www.kaggle.com/datasets/vipullrathod/fish-market): ငါးရဲ့ အလျား၊ အနံ၊ အမြင့်တို့အပေါ် မူတည်ပြီး ငါးရဲ့အလေးချိန် (Weight) ကို ခန့်မှန်းရတဲ့ စိတ်ဝင်စားဖို့ကောင်းတဲ့ Dataset ဖြစ်ပါတယ်။


#### Logistic Regression
- [[ Titanic Dataset ]](https://www.kaggle.com/datasets/heptapod/titanic): လူသိအများဆုံး Dataset ဖြစ်ပါတယ်။ သင်္ဘောနစ်တဲ့အချိန်မှာ ဘယ်သူတွေ အသက်ရှင်မလဲ (Survived: 1 or 0) ဆိုတာကို ခန့်မှန်းရမှာပါ。

- [[ Credit Card Fraud Detection Dataset ]](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud): ငွေပေးချေမှုတစ်ခုဟာ ပုံမှန်လား (Normal) ဒါမှမဟုတ် လိမ်လည်မှု (Fraud) လားဆိုတာကို ခွဲခြားဖို့ပါ。

- [[ Pima Indians Diabetes Dataset ]](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database): လူနာတစ်ယောက်မှာ ဆီးချိုရောဂါ ရှိမရှိ (Outcome) ကို စစ်ဆေးဖို့ သုံးပါတယ်。

- [[ Breast Cancer Wisconsin Diagnostic Dataset ]](https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data): ကင်ဆာအကျိတ်က စိုးရိမ်ရတာလား (Malignant) ဒါမှမဟုတ် ရိုးရိုးလား (Benign) ဆိုတာကို ခွဲခြားရမှာပါ。

- [[ Heart Disease Dataset ]](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset): လူနာရဲ့ ကျန်းမာရေးအချက်အလက်တွေကြည့်ပြီး နှလုံးရောဂါ ဖြစ်နိုင်ခြေ ရှိမရှိကို ခန့်မှန်းရပါတယ်。

- [[ Customer Churn Prediction Dataset ]](https://www.kaggle.com/datasets/rashadrmammadov/customer-churn-dataset): ဘဏ် ဒါမှမဟုတ် တယ်လီကွန်းသုံးစွဲသူတွေက ဝန်ဆောင်မှုကို ဆက်သုံးမှာလား၊ ရပ်ဆိုင်းတော့မှာလား (Churn) ဆိုတာကို ခွဲခြားဖို့ပါ。

#### နမူနာ code
- [[ Social Network Ads Dataset ]](https://www.kaggle.com/datasets/rakeshrau/social-network-ads): လူတစ်ယောက်ရဲ့ အသက်နဲ့ ဝင်ငွေအပေါ် မူတည်ပြီး လူမှုကွန်ရက်ပေါ်က ကြော်ငြာကို ကြည့်ပြီး ပစ္စည်းဝယ်မဝယ် ခန့်မှန်းရတာ ဖြစ်ပါတယ်。
